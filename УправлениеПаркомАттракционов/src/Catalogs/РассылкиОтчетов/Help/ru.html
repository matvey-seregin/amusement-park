Предназначена для управления рассылками отчетов.

Рассылка отчетов позволяет настроить автоматическую отправку сформированных отчетов по расписанию на указанные адреса электронной почты, в файловую систему или в другие приложения.

Команда для работы с рассылками отчетов находится в разделе Администрирование - Печатные формы, отчеты и обработки.

Рассылки отчетов

В форме доступна кнопка + Создать для создания новой рассылки отчетов.

Отборы (фильтры) в списке рассылок

В верхней части формы расположены элементы управления для фильтрации отображаемых рассылок:

Показать

С помощью переключателя или выпадающего списка Показать можно выбрать статус отображаемых рассылок:

  • Все рассылки — отображаются все рассылки независимо от их статуса выполнения.
  • Выполненные — отображаются только рассылки, которые уже были выполнены (отправлены).
  • Невыполненные — отображаются только рассылки, которые еще не были выполнены (находятся в ожидании).

Ответственный

  • Поле Ответственный позволяет отфильтровать рассылки по ответственному пользователю.
  • Доступные значения:
    • Все — отображаются рассылки с любым ответственным.
    • Конкретный пользователь — отображаются только рассылки, где ответственным указан выбранный пользователь (выбирается из списка Пользователи).

Отчет

  • Поле Отчет позволяет отфильтровать рассылки по типу или наименованию отчета.
  • Доступные значения:
    • Все — отображаются рассылки для любых отчетов.
    • Конкретный отчет — отображаются только рассылки, настроенные для выбранного отчета.

Список рассылок

В табличной части отображается перечень рассылок с их наименованиями:

  • Личные рассылки — группа или отдельная рассылка, настроенная для конкретного пользователя.
  • Демо: Архивация в папку файлов — рассылка для архивации отчетов в папку файлов.
  • Демо: Персонализированная по получателям (HTML, NOZ...) — рассылка с персонализированными отчетами для различных получателей.

Действия со списком

  • Кнопка Создать группу — позволяет создать группу для объединения нескольких рассылок.
  • Кнопка Сохранить — сохраняет текущие настройки фильтрации и отображения списка рассылок.
